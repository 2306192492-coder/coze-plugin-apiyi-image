## 代码支持情况

✅ **插件代码已完整支持所有 16 个模型**

代码中已实现：
1. **Nano Banana 系列特殊参数**: `generate_images: True`
2. **Flux 系列宽高比转换**: `aspect_ratio: "1:1"` 等
3. **DALL-E 3 特殊参数**: `quality: "hd"`, `style: "vivid"`, `n: 1`
4. **GPT-Image Mini 限制**: `n: min(n, 2)`
5. **图生图编辑接口**: 支持 `image_url` 和 `mask_url`

**失败原因不是代码问题，而是 API 分组/权限限制**。

## 解决方案

### 方案 1: 升级 API 分组 (推荐)
联系 APIYI 客服，申请升级到以下分组：
- `NanoBananaEnterprise` - 解锁 Nano Banana Pro/2
- `GPTImageAccess` - 解锁 GPT Image 1/1-Mini
- `PremiumModels` - 解锁 Recraft/Ideogram/Midjourney

### 方案 2: 使用已开通模型
当前可用的 5 个模型已经覆盖主要场景：
- **电商/批量**: Nano Banana
- **艺术插画**: Flux Dev
- **图生图编辑**: Flux Kontext Pro/Max
- **超低价头像**: Sora Image

### 方案 3: 等待配额重置
部分模型可能是临时配额不足，可等待次日重试。
