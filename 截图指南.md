# APIYI Coze 插件 - 运行效果截图指南

## 📸 方案 1: 在 Coze 中实际运行（推荐）

### 步骤 1: 创建 Bot
1. 访问 https://www.coze.cn
2. 点击「创建 Bot」
3. 输入名称：`APIYI 图像生成助手`

### 步骤 2: 添加插件
1. 在 Bot 编辑页面，找到「插件」区域
2. 点击「添加插件」→「自定义插件」
3. 上传 `manifest.json` 文件
4. 上传 `handler.js` 文件

### 步骤 3: 配置凭证
1. 点击「凭证」→「添加凭证」
2. 名称：`apiyi_api_key`
3. 类型：`string`
4. 填入你的 API Key：`sk-Z39eHn58dCBzvt0R0b04Ef62C0404003Aa22E992E58f57D8`
