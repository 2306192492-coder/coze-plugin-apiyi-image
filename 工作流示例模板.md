# APIYI 图像生成插件 - 工作流示例模板

## 模板 1: 电商产品图生成 (Nano Banana)

**场景**: 为电商产品生成高质量展示图

**配置**:
```json
{
  "mode": "text2img",
  "model": "gemini-2.5-flash-image-preview",
  "size": "1024x1024",
  "n": 4
}
```

**提示词示例**:
```
白色背景，一个精致的陶瓷咖啡杯，45 度角拍摄，柔和的影棚灯光，
高分辨率产品摄影，细节清晰，适合电商主图
```

**预期输出**: 4 张不同角度的产品图，适合电商详情页使用

**成本**: ~$0.08 (4 张 × $0.02/张)

## 使用说明

### 在 Coze 中使用

1. 在对话中直接描述需求，例如：
   - "帮我生成 4 张电商产品图，白色背景的咖啡杯"
   - "需要一个 YouTube 封面，赛博朋克城市夜景"
   - "把这张照片转成梵高风格" + [上传图片]

2. 插件会自动选择合适的模型和参数

3. 如需指定模型，可以说：
   - "用 Nano Banana 生成..."
   - "用 Flux Kontext Pro 生成..."

### 在代码中使用

```python
import requests

API_KEY = "sk-YOUR_API_KEY"
BASE_URL = "https://api.apiyi.com/v1/images/generations"

# 电商产品图示例
response = requests.post(
    BASE_URL,
    headers={"Authorization": f"Bearer {API_KEY}"},
    json={
        "model": "gemini-2.5-flash-image-preview",
        "prompt": "白色背景，精致的陶瓷咖啡杯，产品摄影",
        "n": 4,
        "size": "1024x1024"
    }
)

images = response.json()["data"]
for img in images:
    print(img["url"])
```

## 模型选择建议

| 场景 | 推荐模型 | 原因 |
|------|----------|------|
| 电商产品 | Nano Banana | 速度快，成本低，适合批量 |
| 视频封面 | Flux Kontext Pro | 支持 16:9，电影质感 |
| 头像/图标 | Sora Image | 超低价，适合大量生成 |
| 艺术插画 | Flux Dev | 艺术表现力强 |
| 风格转换 | Flux Kontext Max | 图生图质量最高 |
| 手机壁纸 | Nano Banana | 支持竖版，成本低 |
